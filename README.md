**Habits Tracker** is a comprehensive Full Stack application developed as the **final graduation project** for the 4Geeks Academy Bootcamp. It is designed to help users register, monitor, and achieve healthy lifestyle goals focusing on sleep, nutrition, and physical activity.

<br>

## Features

- **Daily Habit Logging:** Simple interface to track sleep hours, meals, and exercise.
- **Visual Analytics:** Dynamic charts and statistics to visualize progress over time.
- **Goal Setting System:** Set weekly targets and personal records (e.g., "Run 20km").
- **Interactive Calendar:** A dedicated view to review and manage past habit entries.
- **Secure Authentication:** User accounts protected with **JWT (JSON Web Tokens)**.

<br>

## Technical Highlights (Full Stack)

- **Backend:** RESTful API built with **Python & Flask**.
- **Frontend:** Reactive User Interface developed with **React.js** and state management via **Context API/Redux**.
- **Database:** Relational data modeling using **SQLAlchemy**.
- **Security:** Implementation of password hashing and protected private routes.
- **API Integration:** Integration with third-party services and internal data flows.

## Installation & Usage

1. Clone the repository
   ```bash
   git clone https://github.com/Antonio-Borrero/proyecto-final-4Geeks.git
   ```
   
2. Backend Setup (Python/Flask):
   ```bash
   pipenv install
   pipenv run start
   ```
   
3. Frontend Setup (React):
   ```bash
   npm install
   npm run start
   ```
