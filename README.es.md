**Habits Tracker** es una aplicación Full Stack integral desarrollada como **proyecto final de graduación** para el Bootcamp de 4Geeks Academy. Está diseñada para ayudar a los usuarios a registrar, monitorear y alcanzar metas de vida saludable, centrándose en el sueño, la nutrición y la actividad física.

<br>

## Funcionalidades Principales

- **Registro Diario de Hábitos:** Interfaz sencilla para trackear horas de sueño, comidas y ejercicio.
- **Analítica Visual:** Gráficos y estadísticas dinámicas para visualizar el progreso en el tiempo.
- **Sistema de Metas:** Configuración de objetivos semanales y récords personales (ej: "Correr 20km").
- **Calendario Interactivo:** Vista dedicada para revisar y gestionar entradas de hábitos pasadas.
- **Autenticación Segura:** Cuentas de usuario protegidas mediante **JWT (JSON Web Tokens)**.

<br>

## Aspectos Técnicos Destacados (Full Stack)

- **Back-end:** API RESTful construida con **Python y Flask**.
- **Front-end:** Interfaz de usuario reactiva desarrollada con **React.js** y gestión de estado mediante **Context API/Redux**.
- **Base de Datos:** Modelado de datos relacionales utilizando **SQLAlchemy**.
- **Seguridad:** Implementación de hashing de contraseñas y rutas privadas protegidas.
- **Integración de API:** Comunicación fluida entre front-end y back-end mediante peticiones asíncronas.

<br>

## Instalación y Uso

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Antonio-Borrero/proyecto-final-4Geeks.git
   ```
   
2. Configuración del Backend (Python/Flask):
   ```bash
   pipenv install
   pipenv run start
   ```
   
3. Configuración del Frontend (React):
   ```bash
   npm install
   npm run start
   ```
